# Astro Blog Starter - CR Ferramentas & Cia

Este é o template base utilizado para o desenvolvimento do blog **CR Ferramentas & Cia**, focado em performance extrema e otimização para motores de busca (SEO).

## 🚀 Tecnologias e Foco
- **Framework:** Astro (Static Site Generator)
- **Performance:** Imagens otimizadas em WebP e baixo tempo de carregamento.
- **SEO:** Estrutura focada em Web Vitals e acessibilidade.
- **Hospedagem:** Configurado para alta disponibilidade via Cloudflare.

## 🛠️ Objetivo
Desenvolver uma plataforma rápida e eficiente para compartilhar conteúdo técnico sobre manutenção industrial, ferramentas e mecânica de precisão.
